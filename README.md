
### ps

ps 문제풀이를 위한 레포지스터리입니다.
푼 혹은 풀고 있는 문제들에 대한 코드를 볼 수 있습니다.

ps 연습을 위해 진행하는 것이므로 풀이 스타일이 고정되어있지 않아 문제마다 사용하는 입출력 방식등이 다를 수 있습니다.

# 1.BOJSTEP

[백준 단계별로 풀어보기](https://www.acmicpc.net/step)의 문제들의 풀이를 공유하기 위해 만들어진 레포지스터리입니다.

현재 상황
* 문제풀이 진행중

사용 언어
* c++17

# 2.BOJOTHER

BOJSTEP에 속하지 않는 문제들의 풀이를 공유하기 위해 만들어진 레포지스터리입니다.

현재 상황
* 문제풀이 진행중

사용 언어
* c++17
